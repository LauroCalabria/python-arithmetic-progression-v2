# Arithmetic Progression Generator

## 📌 Description

This project is a Python program that generates an arithmetic progression (AP) based on user input.
The user provides the first term and the common difference, and the program displays the first 10 terms of the sequence using a `while` loop.

## 🚀 Features

* Generates an arithmetic progression
* Accepts user input for first term and common difference
* Displays the first 10 terms of the sequence
* Uses a `while` loop for iteration

## 🛠️ Technologies

* Python

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/LauroCalabria/python-arithmetic-progression-v2.git
```

2. Navigate to the folder:

```bash
cd arithmetic-progression-v2
```

3. Run the file:

```bash
python main.py
```

## 🧪 Example

```bash
Enter the first term: 2
Enter the common difference: 3

Output:
2 5 8 11 14 17 20 23 26 29
```
